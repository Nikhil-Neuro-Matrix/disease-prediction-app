# Disease Prediction System

A Streamlit web application for predicting diabetes risk and classifying skin diseases using machine learning.

## Features
- 🧪 Diabetes Risk Prediction
- 🌿 Skin Disease Classification
- 📸 Image upload and camera capture

## Models Used
- Logistic Regression for Diabetes Prediction
- CNN for Skin Disease Classification

## Disclaimer
This app is for educational and demonstration purposes only. Not for actual medical diagnosis.
